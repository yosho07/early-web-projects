# HTML & CSS Projects

This repository contains some of my early web development projects created while learning the fundamentals of HTML and CSS.

## Projects

### 1. Paralympic Games Website

A static informational website consisting of **5 interconnected webpages** about the Paralympic Games. The project demonstrates:

* Multi-page website structure
* Navigation using hyperlinks
* Images and text formatting
* Lists and tables
* Basic CSS styling

### 2. Online Plant Nursery

A static website consisting of **3 webpages** for an online plant nursery. The project includes:

* Product display pages
* Navigation between pages
* Forms and basic user interface elements
* CSS styling for layout and presentation

## Technologies Used

* HTML5
* CSS3
